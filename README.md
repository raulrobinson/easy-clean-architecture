# Easy Clean Architecture Pattern

This is a simple example of how to implement the Clean Architecture Pattern in Spring Boot.

## Technologies
- Java 17
- Spring Boot 3.2.5
- Lombok
- JPA / Hibernate
- H2 Database
- Maven
- Docker
- Docker Compose
- JUnit 5
- Mockito

## How to run

### Docker Compose
```shell
docker-compose up
```

### Maven
```shell
mvn clean install
mvn spring-boot:run
```
